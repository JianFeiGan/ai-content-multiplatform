# ai-content-multiplatform

AI 内容多平台适配与发布工具 - 一键将核心内容适配到微信公众号、抖音、CSDN、知乎、小红书等平台

## 安装

```bash
pip install ai-content-multiplatform
```

## 使用

```bash
ai-content-multiplatform --help
```

## 开发

```bash
# 克隆项目
git clone git@github.com:JianFeiGan/ai-content-multiplatform.git
cd ai-content-multiplatform

# 安装依赖
pip install -e ".[dev]"

# 运行测试
pytest
```
